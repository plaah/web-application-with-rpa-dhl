Standard Response Envelope (every endpoint):
```json
{ "success": true, "data": {...}, "message": "string" }
```
Errors raise HTTPException(status_code=N, detail="string"); FastAPI returns it as {"detail": "..."}. The frontend tolerates both shapes.

JWT Token Payload (HS256, 8-hour expiry):
```python
{ "sub": user_id, "role": "admin"|"editor", "username": "...", "exp": <utc> }
```


```python
{ "id": "<uuid4>", "title": str, "description": str, "raw_content": str,
  "status": "draft"|"reviewed"|"published",
  "priority": "low"|"medium"|"high"|"critical",
  "category": "late_delivery"|"address_issue"|"damaged_parcel"|"system_error"|"complaint"|"other",
  "tags": [str], "creator_id": str, "creator_name": str,
  "created_at": iso8601, "updated_at": iso8601,
  "source": "manual"|"rpa"|"ai_assisted",
  "files": [{"file_id": uuid, "filename": str, "file_type": str, "path": str, "uploaded_at": iso}],
  "content_hash": str|None, "version": int,
  "version_history": [{"version": int, "description": str, "status": str, "changed_by": str, "changed_at": iso}],
  "ai_suggestions": null|object }
```

Hash record (data/hashes.json):
```python
{ "hash": "<sha256 hex>", "incident_id": "<uuid>", "created_at": iso8601 }
```

Log record (data/logs.json):
```python
{ "log_id": "<uuid>", "run_id": str, "timestamp": iso8601, "action": str,
  "file_name": str|None, "incident_id": str|None,
  "status": "success"|"failed"|"skipped", "message": str,
  "screenshot_path": str|None }
```

```python
VALID_TRANSITIONS = { "draft": "reviewed", "reviewed": "published" }
```
